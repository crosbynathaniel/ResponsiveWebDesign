Introduction

C/C++ are two languages used for the development of computer programs. They are among the most widely used programming languages, and certainly some of the oldest continuously-used languages.

Why Learn C and C++?

People choose to learn C and C++ for a variety of reasons. The most common reasons are:

  • Learning materials are widely available

  • They're powerful and fast

  • Employers are always looking to hire C/C++ programmers.

C versus C++

There are several differences between C and C++. C++ is based around object-oriented programming using classes. Classes are objects that can hold and manipulate data using variables and functions.

C relies on functional programming, and so classes are not present. However, C does have what's called a data structure, which is represented as struct. A struct is not able to use functions within itself, but it can still store data.

Basics

Let's go on a brief dive into the basics of C/C++.

All C/C++ programs must have the main() function. This function is, true to its name, the main place your program runs. A typical main() function looks like this:

int main(int argc, char** argv[]);

The arguments in parenthesis are optional.

C/C++ programs, like most programming languages, have variables, functions, and the aforementioned structs and classes. Since C and C++ are strongly-typed languages, every variable's type has to be declared during initialization, like so.

int number = 11;

There are functions where you can code outside the confines of int main(). You do this by first declaring the return type (what the function outputs) and then typing the function name. An example would be:

void doStuff();

There are also classes, which look like:

class CExample
{
public:
CExample();
~CExample();
private:
int iExampleNumber;
int calcExampleNumber();
}

For structs, it would look like:

typedef struct SExample {
int iExampleNum;
}

There are also preprocessor directives, which can be used to include other C/C++ files, define constants, or to provide messages to the compiler.

With all this said, let's see how a simple "Hello World!" program would look like in C/C++.

For C:

#include <stdio.h>
int main()
{
printf("Hello World!")
}

And in C++

#include <iostream.h>
int main()
{
std::cout << "Hello World!" << endl;
}

Libraries

C/C++ have two different libraries to use, depending on which language you are working with.

  • The C Standard Library
  • The Standard Template Library (STL)

The C Standard Library relies on functions, structs, and preprocessor definitions to provide helpful ways to make applications.

In a similar vein to the Standard Library, the Standard Template Library, made for C++, uses classes and templates to provide helper objects like vector, queues, stacks, and deques.
